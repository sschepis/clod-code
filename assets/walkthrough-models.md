# Choose Your Intelligence

Clod Code is model-agnostic. You can switch between the best models for the job:

- **Anthropic Claude 3.5 Sonnet:** Excellent for complex reasoning and large refactors.
- **OpenAI GPT-4o:** Fast and highly capable.
- **Google Gemini 2.0:** Massive context windows.
- **Local Models:** Connect via LM Studio or Ollama for privacy.

Set your API keys securely in the Clod Code settings.
