# Build Interactive Tools

Sometimes a chat message isn't enough. Oboto VS can build and deploy **Surfaces** and **Routes** on the fly.

- **Surfaces:** Interactive HTML/JS panels that render inside VS Code. Great for dashboards, visualizations, or custom forms.
- **Routes:** Local Next.js-style API endpoints that run on a background server.

Ask Oboto VS to "Build a dashboard surface to visualize my database" and watch it happen!
